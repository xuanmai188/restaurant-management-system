# Hệ thống quản lý nhà hàng

## Giới thiệu
Dự án xây dựng hệ thống quản lý nhà hàng nhằm hỗ trợ quản lý các hoạt động cơ bản như quản lý bàn, menu món ăn, order, hóa đơn và đặt bàn.

## Thành viên nhóm
- Thành viên 1: Nguyễn Ngọc Xuân Mai
- Thành viên 2: Huỳnh Văn Mới
- Thành viên 3: Trần Thị Thu Lan

## Chức năng chính
- Đăng nhập hệ thống
- Quản lý bàn
- Quản lý menu món ăn
- Tạo order cho bàn
- Lập hóa đơn thanh toán
- Tạo phiếu đặt bàn

## Công nghệ dự kiến
- Frontend: HTML/CSS/JavaScript hoặc ReactJS
- Backend: Node.js / Spring Boot
- Database: MySQL / SQL Server
- Quản lý công việc: Jira
- Quản lý mã nguồn: GitHub
